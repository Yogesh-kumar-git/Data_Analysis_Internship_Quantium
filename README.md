# 📊 Quantium Data Analytics Virtual Internship (Forage)

## 📌 Project Overview

This repository showcases my work completed as part of the **Quantium Data Analytics Virtual Internship** hosted on Forage.

The project simulates a real-world retail analytics scenario, focusing on **chip purchase behavior in supermarkets**. The analysis aims to understand customer segments, evaluate purchasing patterns, and assess the effectiveness of a **store layout trial**.

The end goal was to generate **data-driven insights and business recommendations** for strategic decision-making.

---

## 🎯 Key Objectives

* Analyze customer purchasing behavior
* Identify high-value customer segments
* Evaluate product and sales performance
* Measure the impact of store layout trials
* Deliver actionable business recommendations


## 🔍 Task 1: Data Preparation & Customer Analytics

### ✅ Data Cleaning

* Converted date from integer to datetime format
* Removed irrelevant categories (e.g., salsa products)
* Handled outliers and inconsistencies

### 📊 Analysis Performed

* Sales analysis by customer segments:

  * **LIFESTAGE**
  * **MEMBER_TYPE**
* Identified high-performing customer groups
* Deep dive into **Mainstream Young Singles/Couples** segment

### 💡 Key Insights

* Top contributing segments:

  * Budget Older Families
  * Mainstream Young Singles/Couples
  * Mainstream Retirees

* Older families purchase the **highest number of packets per customer**

* Mainstream young singles/couples represent the **largest customer base**

* **Kettle chips** and **175g packs** dominate across segments

* Mainstream young singles/couples are:

  * **28% more likely** to purchase Tyrrells chips
  * **32% more likely** to buy 270g packs (e.g., Twisties)

## 🔬 Task 2: Experimentation & Uplift Testing

### 🧪 Problem Statement

Evaluate the impact of a **new store layout trial** conducted in stores:

* 77, 86, 88 (Trial stores)
* Trial Period: Feb – Apr 2019

### ⚙️ Methodology

* Selected control stores using:

  * **Pearson correlation**
  * **Magnitude distance metrics**
* Compared:

  * Total Sales
  * Number of Customers
* Applied **hypothesis testing** to validate significance

### 📊 Key Results

* Control store pairs:

  * Store 77 → 233
  * Store 86 → 155
  * Store 88 → 40

* **Sales Impact:**

  * Store 77 → Significant increase in March & April
  * Store 86 → Increase observed in March

* **Customer Growth:**

  * Stores 77 & 86 showed significant increase
  * Store 88 → No significant impact

### 💡 Conclusion

The trial layout was **successful in stores 77 and 86**, but **ineffective in store 88**, indicating location-specific performance variation.
